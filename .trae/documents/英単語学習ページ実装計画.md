# 英単語学習ページ（ランダム表示）の実装

## 1. ページ構成
- **URL**: `/study`
- **ファイル構成**:
  - `src/app/study/page.tsx`: サーバーコンポーネント。単語データを読み込み、クライアントコンポーネントへ渡す。
  - `src/app/study/StudyClient.tsx`: クライアントコンポーネント。状態管理（現在の単語、ダイアログ表示）とUIイベントを処理。
  - `src/app/study/study.module.css`: 専用スタイル。既存の `page.module.css` のデザインを踏襲。

## 2. 実装詳細

### A. データ取得 (`src/app/study/page.tsx`)
- `src/data/words.ts` の `getAllWords()` を使用して全単語を取得。
- `StudyClient` コンポーネントに `words` プロパティとして渡す。

### B. クライアントロジック (`src/app/study/StudyClient.tsx`)
- **State**:
  - `currentWord`: 現在表示中の単語オブジェクト
  - `showDialog`: ダイアログの表示状態 (boolean)
- **Functions**:
  - `pickRandomWord()`: ランダムに単語を選出する。現在と同じ単語が選ばれないように配慮。
  - `handleRemembered()` (〇ボタン): 次の単語へ更新。
  - `handleForgot()` (×ボタン): `showDialog` を true にする。
  - `handleDialogConfirm()` (はい): `/words/[slug]` へ遷移。
  - `handleDialogCancel()` (いいえ): ダイアログを閉じ、次の単語へ更新。

### C. UIデザイン (`src/app/study/study.module.css`)
- `page.module.css` の `wordCard` スタイルをベースに、中央表示用に調整したカードデザイン。
- ボタンデザインは `pageButton` のスタイルを参考に、大きく押しやすいアクションボタンを作成。
- ダイアログはオーバーレイ（モーダル）として実装し、既存のスタイルガイド（配色、角丸）に合わせる。

## 3. 手順
1. **スタイル作成**: `src/app/study/study.module.css` を作成。
2. **クライアント実装**: `src/app/study/StudyClient.tsx` を実装。
3. **ページ実装**: `src/app/study/page.tsx` を実装。
4. **ナビゲーション追加**: 必要に応じてトップページ等にリンクを追加（今回は要件に含まれないため、直接URLアクセスで確認）。
5. **検証**: 手動テスト（〇/×/ダイアログ遷移）を実施。

## 4. 備考
- 既存の `src/data/words.ts` や `src/app/words/[word]/page.tsx` への変更はありません。
- デザインは `page.module.css` から `radial-gradient` 背景や `wordCard` のシャドウ・ボーダーなどの変数をコピー/参照し、一貫性を保ちます。
